How to run:
     Windows:
         * Please run the "scripts/init.bat" script to ensure every dependencies is installed (before everything)
         * To start the project, from the parent directory use the following command "python3 scripts/run.py"
     Linux/MacOS:
         * Please run the "scripts/init.sh" script to ensure every dependencies is installed (before everything)
         * To start the project, from the parent directory use the following command "python3 scripts/run.py"
